Organizing with Tags

Tags let you group related launchd jobs with colored labels. This is useful when you manage many jobs and want to quickly filter to a specific set — for example, all jobs related to a particular project.

Creating Tags

Open Settings (⌘,) and go to the Tags tab. Type a name for your new tag, select a color from the preset palette, and click Add.

Renaming Tags

In the Tags settings tab, click the pencil icon next to a tag name. Edit the name in the text field and click Done. Renaming a tag does not affect which jobs it's assigned to — tags use stable identifiers independent of their display name.

Changing Tag Colors

In the Tags settings tab, click any color dot below a tag to change its color. The change takes effect immediately across the entire app.

Assigning Tags to Jobs

You can assign tags in two ways:

  • Context menu — Right-click a job in the sidebar and use the Tags submenu to toggle tags on or off.
  • Detail pane — In the job detail view, click the tag button in the header to assign or remove tags.

A job can have multiple tags. Tag badges appear on the job row in the sidebar and in the detail view header.

Filtering by Tag

Use the tag picker in the toolbar to filter the sidebar to jobs with a specific tag. Active tag filters appear as colored chips in the filter status bar at the bottom of the sidebar.

Default Tag Filter

In Settings → General, you can set a default tag filter that's applied automatically when Taktwerk launches.

Deleting Tags

In the Tags settings tab, click the trash icon next to a tag. The tag is removed from all jobs and deleted permanently.
