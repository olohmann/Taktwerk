Browsing & Filtering Jobs

The Sidebar

The sidebar shows all launchd jobs found on your Mac. Each row displays the job label, its current status (loaded or not), and any assigned tags.

Source Filter

Filter jobs by where they come from:

  • All — Show every job from all directories.
  • User Agents — Jobs in ~/Library/LaunchAgents/.
  • Global Agents — Jobs in /Library/LaunchAgents/.
  • Global Daemons — Jobs in /Library/LaunchDaemons/.

Change the source using the picker in the toolbar.

Tag Filter

If you've assigned tags to jobs, you can filter the list to show only jobs with a specific tag. Select a tag from the tag picker in the toolbar.

Filter Status Bar

When any filter is active, a status bar appears at the bottom of the sidebar showing colored chips for each active filter. Click the × on a chip to remove that filter, or click Clear to reset all filters.

Searching

Use the search field at the top of the sidebar to find jobs by label. The search filters the current view in real time.

Refreshing

To re-scan all launchd directories and pick up external changes (e.g., files added or removed outside Taktwerk), use ⌘R or click the refresh button in the toolbar.
