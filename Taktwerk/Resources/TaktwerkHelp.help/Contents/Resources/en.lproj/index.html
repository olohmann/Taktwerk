Taktwerk Help

A native macOS app for managing launchd agents and daemons.

  • Browsing & Filtering Jobs Navigate launchd sources, search jobs, and use filters to find what you need.
  • Editing Plist Files Use the visual form editor or raw XML editor with live validation.
  • Organizing with Tags Create colored tags, assign them to jobs, and filter by tag.
  • Settings & Preferences Configure default filters, manage tag definitions, and customize your experience.
  • Loading & Unloading Jobs Control launchd jobs directly from the app — load, unload, and check status.

Getting Started

Taktwerk scans all standard launchd directories on your Mac and displays every agent and daemon it finds. Use the sidebar to browse jobs, the detail pane to inspect them, and the editor to make changes.

Tip: Use ⌘R to refresh the job list after making changes outside the app.

Launchd Directories

Taktwerk monitors these standard macOS directories:

  • User Agents — ~/Library/LaunchAgents/
  • Global Agents — /Library/LaunchAgents/
  • Global Daemons — /Library/LaunchDaemons/
