Settings & Preferences

Open Settings with ⌘, or from the Taktwerk menu.

General Tab

Default Source Filter

Choose which source filter is applied when Taktwerk launches:

  • All
  • User Agents
  • Global Agents
  • Global Daemons

Default Tag Filter

Choose a tag to filter by on launch, or select All to show every job regardless of tags.

Tags Tab

Manage your tag definitions:

  • Add tags — Enter a name, pick a color, click Add.
  • Rename tags — Click the pencil icon, edit the name, click Done.
  • Change colors — Click a color dot to change a tag's color.
  • Delete tags — Click the trash icon to remove a tag.

Tags use stable identifiers, so renaming a tag preserves all existing assignments.

Tip: Taktwerk comes with two default tags (Tag-1 and Tag-2) to get you started. Feel free to rename or delete them.
